# Hackathon-2
Qué problema resuelve el sistema Implementa una agenda telefónica que:  guarda contactos (nombre, apellido, teléfono),  evita duplicados,  valida datos,  permite operaciones CRUD (agregar, listar, buscar, eliminar, modificar),  respeta un tamaño máximo (por defecto 10),  y se prueba por consola y se demuestra por JavaFX.
